# Musiki Sithomola — Portfolio

Premium 2026 portfolio built with **Next.js 15**, **React**, **TypeScript**, **Tailwind CSS**, **shadcn/ui**, and **Framer Motion**.

## Stack

- Next.js 15 (App Router)
- Geist + Satoshi typography
- Framer Motion animations
- Formspree contact & booking forms
- Lucide icons

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```

## Structure

- `app/` — layout, page, global styles
- `components/sections/` — homepage sections
- `components/layout/` — navbar & footer
- `lib/data.ts` — portfolio content
- `public/` — project images & CV
